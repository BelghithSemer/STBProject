Project Description: Bank Dashboard

The Bank Dashboard is an advanced platform designed to provide a seamless and efficient banking experience. It includes the following features:

1. Simulation for Credits: Users can simulate various credit scenarios to understand potential financial outcomes and make informed decisions.

2. Requesting a New Credit: This feature allows users to request new credits directly from the dashboard, streamlining the credit application process.

3. Credit Card Transactions Management: Users can manage and track their credit card transactions, providing a clear overview of their spending and financial activity.

4. Statistics: The dashboard offers detailed statistical insights and visualizations to help users understand their financial data and trends.

5. Profile Management: Users can manage their personal information and account settings through an intuitive profile management system.

The project is developed using .NET for the backend and Angular for the frontend, ensuring a robust and user-friendly interface.
